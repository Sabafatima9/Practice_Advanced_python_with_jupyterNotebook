## 📖 About This Repository

This repository continues my Python learning journey beyond the basics — diving into **file handling, data formats, and error handling**. Each notebook builds on the previous one, moving from reading plain text files to working with structured data (CSV, JSON) and writing robust, error-safe code.

> 💡 *"Real-world Python isn't just about syntax — it's about handling real data and real errors gracefully."*

---

## 🗂️ What's Inside

| # | Topic | Description |
|---|---|---|
| 01 | 📂 **Introduction to File Handling** | Opening, reading, writing & closing files in Python |
| 02 | 📄 **Working with TXT Files** | Reading/writing plain text files, modes (`r`, `w`, `a`) |
| 03 | 📊 **Working with CSV Files** | Reading & writing tabular data using the `csv` module |
| 04 | 🧾 **Working with JSON Files** | Parsing and creating JSON data with the `json` module |
| 05 | ⚠️ **Exceptions & Error Handling** | `try` / `except` / `finally`, custom exceptions |
